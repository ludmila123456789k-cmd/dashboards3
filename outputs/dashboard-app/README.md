# Dashboard Studio

Минималистичный веб-дашборд для показателей сайта, лидов, продаж, маркетинга и KPI руководителя.

## Запуск

```powershell
.\start-dashboard.ps1
```

После запуска откройте:

```text
http://localhost:4173
```

## Ссылка руководителю

Кнопка `Ссылка руководителю` копирует публичную HTTPS-ссылку просмотра, если приложение запущено на публичном домене или задан `PUBLIC_VIEW_BASE_URL`.

Локальный адрес `localhost` подходит только для вашего компьютера. Для просмотра с другого Wi-Fi нужен обычный публичный сервер или хостинг с HTTPS.

Редактирование по ссылке руководителя недоступно: она открывает только режим просмотра.
